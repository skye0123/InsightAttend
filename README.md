# InsightAttend
**LAN-Based Employee Attendance System with Facial Recognition and Predictive Analytics (Prototype)**  

This repository contains the 50% version of the system for demonstration and development in C# (Windows Forms).

## Features (so far)
- Basic Login (admin/admin)
- Main Dashboard UI
- Placeholder Employee Management and Reports
- DataAccess class ready for MS Access integration

## Next Steps
- Connect to `InsightAttend.accdb`
- Add Employee CRUD Forms
- Add Facial Recognition (using Emgu CV)
- Add Predictive Analytics (attendance trends)
